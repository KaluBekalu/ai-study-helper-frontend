## ğŸ“¦ Installation

Follow these steps to set up the project locally:

1. Clone this repo like a boss:

   ```bash
   git clone https://github.com/kalubekalu/ai-study-app.git
   cd ai-study-app
   ```

2. Install dependencies:

   ```bash
   npm install
   ```


